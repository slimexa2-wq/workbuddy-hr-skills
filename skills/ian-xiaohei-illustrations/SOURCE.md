# Ian Xiaohei Illustrations

这是从开源项目 `helloianneo/ian-xiaohei-illustrations` 保存的可安装 Skill 核心文件。

## 原项目定位

将中文文章、帖子、博客、Notion 文档、工作流文档和方法论内容中的关键判断、流程、结构、状态或隐喻，转化为 16:9 白底手绘“小黑”正文配图。

## 使用说明

安装时复制整个 `ian-xiaohei-illustrations` 目录到目标 Agent 的 Skills 目录。

Codex 示例：

```bash
mkdir -p "${CODEX_HOME:-$HOME/.codex}/skills"
cp -R ./skills/ian-xiaohei-illustrations "${CODEX_HOME:-$HOME/.codex}/skills/"
```

调用示例：

```text
Use $ian-xiaohei-illustrations 为这篇中文文章设计并生成 5 张小黑怪诞正文配图。
```
